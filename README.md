# Forma - Agency Landing Page Template

A premium, modern landing page template designed for digital agencies, creative studios, and software companies. Built with React, Vite, and Tailwind CSS.

## 🚀 Features

- **Responsive Design**: Flawlessly adapts to any screen size (mobile, tablet, desktop).
- **Video Background**: Immersive hero section with an auto-playing video background.
- **Glassmorphism UI**: Beautiful semi-transparent frosted glass elements.
- **Modern Routing**: Clean navigation between the Home and "Our Story" pages using `react-router-dom`.
- **Contact Form UI**: Ready-to-use aesthetic form component for capturing leads.

## 🛠️ Tech Stack

- **React 18**
- **Vite**
- **Tailwind CSS**
- **TypeScript**
- **Lucide Icons**
- **Bun** (Package Manager)

## 📦 Getting Started

### Prerequisites
Make sure you have [Bun](https://bun.sh/) installed on your machine.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Yugha13/landing-page-template.git
cd agency-landing-page
```

2. Install dependencies:
```bash
bun install
```

3. Start the development server:
```bash
bun dev
```

4. Open your browser and navigate to the local URL provided in your terminal (usually `http://localhost:5173`).

## 📁 Project Structure

- `src/components/Layout.tsx` - Main layout wrapper including the video background and navigation bar.
- `src/pages/Home.tsx` - The primary landing page featuring the headline and contact form.
- `src/pages/Story.tsx` - The "Our Story" page detailing the agency's origins.
- `src/App.tsx` - Application entry point and router configuration.
